# BoardNest
Kanban-style task management web app that enables users to organize workflows, visualize task progress through customizable boards.
# Tech Stack

- **Node.js**
- **TypeScript**
- **Express.js**
- **Swagger**
- **CORS**
- **jsonwebtoken**
- **React**
- **Tailwind**
- **JSX**
- **Sequelize**

# Installation

## Backend

1. **Clone the repository.**

   ````bash
   git clone https://github.com/ismaelvr1999/boardnest.git
   ````

2. **Install dependencies.**

   ````bash
   cd  boardnest
   cd  backend
   pnpm install
   ````

3. **Set up environment variables.**
Create a .env file add the following variables:
   ```env
   DB_HOST = 
   DB_USER = 
   DB_PASSWORD = 
   DB_NAME = 
   FRONTEND_URL = 

   PORT = 
   JWT_SECRET = 
   ```
4. **Start the server.**
    ```bash
    pnpm run dev
    ```
## Frontend

1. **Install dependencies.**

   ````bash
   cd  boardnest
   cd  frontend
   pnpm install
   ````

2. **Set up environment variables.**
Create a .env file add the following variables:
   ```env
   VITE_API_URL = 
   ```
3. **Start the client.**
    ```bash
    pnpm run dev
    ```
